# Sistema Escolar - MVC

Este projeto é um sistema backend desenvolvido com Node.js utilizando o padrão MVC (Model-View-Controller) e integração com banco de dados.

## Funcionalidades

- Cadastro de alunos
- Listagem de dados
- Estrutura organizada em MVC
- Integração com banco de dados

## Tecnologias

- Node.js
- JavaScript
- Banco de Dados (ex: MySQL ou SQLite)

## Estrutura

- `models` → regras de negócio e banco
- `controllers` → lógica da aplicação
- `views` → interface (se tiver)
